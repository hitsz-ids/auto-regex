## pip安装

推荐使用 pip 命令进行安装：

```bash
pip install auto-regex
```

将从[PyPI](https://pypi.org/)获取并安装最新的稳定版本。



## 源码安装

auto-regex源码可以从 Github项目下载，然后执行安装脚本，可通过在终端中执行以下命令安装：

```bash
git clone https://github.com/hitsz-ids/auto-regex.git
cd auto-regex
python setup.py
```
